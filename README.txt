
Software Installer
Build by AKBBProject
Published by ComputerPoint

Requirements:
- Windows 10/11
- Internet connection
- winget (Windows Package Manager)

How to run:
1. Extract the ZIP file.
2. Double-click run.bat.
3. The script will check Python and install it if missing.
4. Use the menu to search, install, uninstall, or update software.
